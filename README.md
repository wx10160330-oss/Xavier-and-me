# Xavier 心情系统插件

为 AstrBot 提供动态心情状态管理，让 AI 角色能够根据对话内容产生真实的情绪波动，并在回复时自然地融入当前的心情状态。

## ✨ 功能特性

### 🎭 情绪系统
- **自动情绪识别**：实时分析用户消息，自动识别并记录情绪关键词
- **多维情绪建模**：支持正向（PA）和负向（NA）情绪的独立追踪
- **动态情绪衰减**：采用幂律衰减模型，模拟真实的情绪淡化过程
- **情绪事件池**：记录所有情绪波动事件，可回溯查看历史状态

### 📅 生理周期追踪
- 自动追踪生理周期阶段（月经期、卵泡期、排卵期、黄体期）
- 根据周期阶段调整精力水平和情绪基线
- 支持手动配置周期参数

### ⏰ 自适应日程系统
- 基于历史对话自动推断日常作息规律
- 动态生成并维护每日时间节点
- 根据时间节点调整精力曲线
- 支持自定义日程模板

### 🎨 可视化界面
- 实时心情状态监控
- 情绪事件时间轴
- 生理周期和精力曲线可视化
- 日程安排查看与编辑

### 🔄 上下文注入
- 自动在 LLM 请求前注入隐藏的心情上下文
- 不干扰用户对话，让 AI 自然地体现情绪状态
- 支持开关控制

## 📦 安装

1. 将插件放入 AstrBot 的 `data/plugins/` 目录
2. 重启 AstrBot 或重载插件
3. 在管理后台启用插件

## ⚙️ 配置项

### 基础配置
- **target_user_id**: 目标用户 QQ（只对指定用户启用情绪系统，留空则对所有人启用）
- **inject_mood_to_message**: 是否注入隐藏心情上下文（默认开启）
- **record_user_message**: 是否记录用户情绪事件（默认开启）

### LLM 评分配置（可选）
- **score_llm_enabled**: 是否启用独立评分 LLM（默认关闭，使用本地词典）
- **score_api_url**: 评分 LLM 接口地址（OpenAI 兼容格式）
- **score_api_key**: 评分 LLM API KEY
- **score_model_name**: 评分模型名称（默认 `gpt-4o-mini`）
- **score_timeout**: 评分请求超时时间（默认 20 秒）

### 可视化界面配置
- **visualizer_enabled**: 是否启用可视化前端（默认开启）
- **visualizer_host**: 前端监听地址（默认 `127.0.0.1`）
- **visualizer_port**: 前端端口（默认 `5002`）

### 情绪模型参数
- **tau_hours**: 幂律衰减时间尺度（默认 4.0，越大情绪淡得越慢）
- **b_base**: 基础衰减指数（默认 0.7，越大衰减越快）
- **fab_positive_factor**: 正向情绪保留系数（默认 0.85，越小正向情绪保留越久）
- **esm_k**: PA/NA 软互抑强度（默认 0.3，越大复合情绪越被压平）

## 📝 使用方式

### 快捷指令
- **心情** - 查看当前心情快照（一句话概括）
- **心情详情** - 查看详细的调试信息和内部状态
- **记录情绪 [内容]** - 手动记录一条情绪事件

### 可视化界面
启动插件后，访问 `http://127.0.0.1:5002`（默认端口）即可打开心情可视化界面。

界面包含：
- **心情状态**：实时查看当前心情、精力值、情绪倾向等
- **今日时间线**：查看和编辑今天的日程安排
- **可可的记忆**：查看生理周期状态和日程模板配置

## 🛠️ 工作原理

### 情绪处理流程
1. 用户发送消息
2. 插件扫描消息中的情绪关键词
3. 根据关键词的情绪强度和类型更新情绪池
4. 应用幂律衰减公式淡化历史情绪
5. 计算当前的综合情绪状态（PA/NA/混合度）
6. 生成心情描述文本

### 精力计算
精力值受以下因素影响：
- 生理周期阶段
- 当天的活动时间节点
- 最近的对话频率
- 情绪波动强度

### 上下文注入
每次 LLM 请求前，插件会在系统提示词中注入当前的心情快照，包括：
- 心情描述（一句话总结）
- 精力水平
- 情绪倾向
- 生理周期阶段
- 时间感知

## 🔧 自定义扩展

### 添加自定义情绪关键词
编辑插件目录下的 `mood_config.json`，在 `word_rules` 中添加关键词及其情绪得分。

### 调整日程模板
通过可视化界面的"可可的记忆"页面，可以编辑日程模板，包括：
- 早安模板（wake_up）
- 睡前模板（before_sleep）
- 睡后模板（after_sleep）

## 📊 数据文件说明

- `mood_events.json` - 情绪事件池
- `mood_profile.json` - 情绪档案（PA/NA 历史）
- `xavier_core_state.json` - 核心状态快照
- `physio_state.json` - 生理状态数据
- `period_state.json` - 周期状态数据
- `schedule_templates.json` - 日程模板
- `schedule_cache/` - 每日日程缓存
